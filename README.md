# Hands-on AWS: CDK with Python exercises

## Prerequisites

- Python 3.11+
- virtualenv
- Poetry

## Setup

1. Create a virtual environment

```bash
python3 -m venv .venv
```

2. Activate virtual environment

```bash
source .venv/bin/activate
```

3. Install dependencies

```bash
poetry install
```
